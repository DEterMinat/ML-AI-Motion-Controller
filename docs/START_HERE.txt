1️⃣  SETUP ENVIRONMENT
    ├─ Windows: Double-click setup.bat (easiest!)
    ├─ Or manually: conda env create -f conda-environment.yml
    └─ Then: conda activate motion-controller

2️⃣  COLLECT TRAINING DATA
    ├─ Run: python data_collector.py
    ├─ Collect 50-100 samples per action (neutral, left_punch, right_punch)
    └─ Output: boxing_data.csv

3️⃣  PLAY THE GAME!
    ├─ Step 2a: python train_model.py (trains model)
    ├─ Step 2b: python main.py (real-time control)
    └─ Your punches control the Roblox game!
